# Bioskop Mini CLI 🎬

Project Python terminal untuk manajemen data film. Dibuat untuk belajar dasar pemrograman.

## ✨ Fitur
- **Tambah Film**: Input judul, genre, tahun rilis
- **Lihat Film**: Menampilkan semua data film yang tersimpan 
- **Cari Film**: Cari film berdasarkan judul
- **Penyimpanan**: Data otomatis tersimpan di file `data_film.txt`

## 🛠️ Teknologi
- Python 3
- List & Dictionary
- File Handling `.txt`

## ▶️ Cara Menjalankan
1. Pastikan sudah install Python 3
2. Buka terminal/cmd
3. Run: `python main.py`
